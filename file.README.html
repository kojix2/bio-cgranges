bio-cgranges

Ruby bindings to lh3/cranges.

cgranges is a small C library for genomic interval overlap queries:

given a genomic region r and a set of regions R, finding all regions in R that overlaps r.

Installation

gem install bio-cgranges

Usage

require "bio/cgranges"

g = Bio::CGRanges.new
g.add("chr1", 10, 20, 0)
g.add("chr1", 15, 25, 1)
g.index()

g.overlap("chr1", 12, 16) do |g|
  p r
end
